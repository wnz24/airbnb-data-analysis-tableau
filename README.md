# Seattle Airbnb Listings — Price & Revenue Analysis

An interactive Tableau dashboard exploring how Airbnb listing prices vary by location, bedroom count, and time across Seattle.

## 📌 Overview
This project analyzes the Airbnb listings dataset (Kaggle / Inside Airbnb) to understand pricing patterns and revenue trends, with a focus on how price scales with bedroom count and how it varies geographically by zipcode.

## 📊 Dashboard Views
- **Price by ZipCode** — average listing price mapped and ranked across Seattle zipcodes
- **Avg Price per Bedroom** — average price trend from studio (0BR) to 6BR listings
- **Revenue for Year** — weekly revenue trend over the calendar year
- **Bedroom Listing Filter** — interactive filter by distinct bedroom count

## 💡 Key Findings
- Price scales consistently with bedroom count, from ~$105 for studios up to ~$585 for 6-bedroom listings
- Pricing varies meaningfully by zipcode — e.g. 98177 averages ~$84/night versus ~$105/night in 98144
- Weekly revenue rose sharply early in the year before leveling off around $2M
